## Part1:部分基础知识

首先我们需要知道，相机是由机身和镜头组成。

### 机身

本次讲解都是基于尼康Z6III机身上的键位来进行讲解，因为个人觉得这台相机确实简洁又方便（虽然我朴素的买不起嘻嘻）。或许当你拿到另外一个相机会存在不同，但是只要学会相关操作，只需要自己尝试一下就可以贯通

位于相机顶部右前方的一个向下按的按钮就是快门，需要知道，这个按钮可以半按进行对焦，全按就是拍下一张照片，很简单，但必须存在。

#### *拍摄模式

看到机身上方，可以找到几种拍摄模式，分别是M、A、S和auto档位。(P档位是啥我咋没用过)

- auto：全自动模式，相机自己根据当前的环境来调节相应曝光参数。
- A：光圈优先模式，也就是相机会调整除了光圈以外的参数来实现你的要求。
- S：快门优先模式，用这个快门速度和相机自动调整的其他参数来实现拍摄。
- M：全手动，也就是你自己调整你想要的需要的参数来达到你预期的摄影效果。

在现实生活中，我们一般会选择前三种模式，在拍摄人像、风光的时候我们会偏向于选择光圈优先模式，在拍摄运动、流水、车流等动态的物体场景下，我们偏向于使用快门优先模式。

#### 画幅（传感器尺寸）

“画幅”指的是相机感光元件（CMOS）的物理尺寸。传感器尺寸影响视角、景深、噪点和动态范围等。在我们的相机中主要用到了半画幅和全画幅两个尺寸（当然还存在更大和更小的，比如富士会选择中画幅做相机，导致镜头很贵很贵；比如手机中和pocket3中会选择更小的画幅，毕竟只有那么点空间）

![cmos](photos/cmos.png)

!!! Warning "不同的是"
	需要注意的是：全画幅的镜头放在半画幅机身上会产生裁剪（视角变窄）。如果你打算以后升级到全画幅，买镜头时要考虑兼容性与预算。
    >（半画幅焦段 = 全画幅 * 1.4左右）

总结一句话就是有钱就买全画幅，没钱就买半画幅。

### 镜头

我手上拿着的镜头叫做 `Z50 1.8s`,我之前还有一个 `Z24-70 f4s`的镜头。这就是定焦镜头和变焦镜头

- `Z`：可以看到刚开始的Z代表的是卡口，需要注意不同相机比如佳能，富士，索尼的微单和单反的镜头卡口都不一样。市面上存在转接环解决这个问题但是会影响性能和素质（对焦速度等）

- `50`和`24-70`：可以看到这代表了焦段，50是定焦，固定不变，而24-70是可以在这个范围内变化，新手可以购入变焦哦。

- `f1.8`和`f4`：这代表了镜头的最大光圈的大小，在部分变焦镜头中其实最大的光圈也是会变的哦。

至于后面的s就是尼康为了彰显镜头素质多好多好吹的。镜头的素质高低会影响成像画质，但是在日常工作摄影中影响其实微乎其微。还有大家尽量不要买二手镜头哦，很多暗病，内灰等。

!!! Tip "大三元和小三元"
    在摄影中我们对三个参数为 `16-35mm f/2.8` `24-70mm f/2.8` `70-200mm f/2.8`的镜头成为大三元，`16-35mm f/4` `24-70mm f/4` `70-200mm f/4`称为小三元。

#### 焦段

![jiaoduan](photos/jiaoduan.png)

焦段决定了镜头的视野和画面表现：焦段越短（广角），视野越广，能包含更多场景；焦段越长（长焦/远摄），视野越窄，但能把远处的主体“拉近”。

- 广角：容易产生畸变（靠近画面边缘会被拉伸），建筑拍摄或需要直线的场景要注意构图或后期矫正。
- 长焦：存在透视压缩，长焦会“压缩”前后空间，看起来前后物体更靠近；。这也是为什么人像常用中长焦来获得更好的人脸比例与背景虚化。

- 景深与背景虚化：在相同构图下（主体大小相同），较长焦段会更容易获得浅景深和背景虚化；相反广角在相同光圈下景深更深。注意构图与拍摄距离都会影响最终效果。

    可以看看这个[微信小视频](photos/video.mp4)让大家理解一下下。


!!! Warning "拍摄注意"
    在某些时候我们手上的相机由于焦段不够导致无法拍全合照，这时候我们不妨拿出手机调节至广角模式拍摄哦！
    
#### 光圈

光圈决定镜头能让多少光进来，常以 f 值表示（例如 f/1.8、f/4、f/8）。f 值越小表示光圈越大，进光量越多；f 值越大表示光圈越小，进光量越少。光圈不仅影响亮度，也直接决定景深和虚化效果。

- 景深与虚化：在相同构图下，光圈越大，景深越浅，背景越虚化；光圈越小，景深越深，背景更清晰。（可以看看后面的example理解）
- 衍射与锐度：光圈开得太小，画面会受衍射影响出现整体锐度下降，所以风光拍摄虽然需要较深景深，也要在锐度和景深之间折中（常用 f/8–f/11）。

### 快门速度

快门速度，可以理解为快门一开一闭的时间差，我们的cmos会保留这段时间内的所有内容然后成为一张照片，这也是可玩性最高的一个点了。同时不难理解快门速度越快，进光量也就越小，我们得到的照片就会越暗，一般日常拍摄时候，1/250的快门速度就差不多，可以避免模糊哦。

比如在运动摄影时，我们需要极快的快门速度以防止模糊，我们也可以特意拉长快门速度来进行一些光绘，还有旋转花？

![flower](photos/flower.jpeg)

### ISO

ISO，即感光度，是相机对光的灵敏度。 拉高ISO可以提亮画面，但是同样会造成很大很大的负面影响，即会产生`噪点`，这对于一张照片来讲是致命的。所以调节ISO的优先级往往是最最低的

### 曝光补偿

曝光补偿，即EV值，它的基本用处其实和ISO差不多，造成的结果也差不多。（我平常不太用到这个参数hhh）
