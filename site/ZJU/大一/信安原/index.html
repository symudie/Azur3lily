    信息安全原理与数学基础¶

    课程概述

    本课程主要涵盖两大模块：离散数学 和 概率论。

    • 离散数学部分理论性较强，需要重点理解概念
    • 概率论部分实用性强，历年真题重复率较高
    • 23届与24届概率论题目相似度很高，建议重点练习历年题

    📋 历年试题分析（来源：cc98）¶

    🎲 概率论部分¶

    一、填空题¶

    题号 题目描述
    第1题 现实场景适用哪种概率模型的判断题
    第2题 条件概率与独立事件
    • 每天有0.4概率有雾
    • 有雾时居民有0.2概率戴口罩
    • 无雾时居民有0.01概率戴口罩

    第2题详细解答：

    求一天中居民戴口罩的概率

    解： 使用全概率公式 \(\(P(\text{戴口罩}) = P(\text{戴口罩}|\text{有雾}) \times P(\text{有雾}) + P(\text{戴口罩}|\text{无雾}) \times P(\text{无雾})\)\) \(\(= 0.2 \times 0.4 + 0.01 \times 0.6 = 0.08 + 0.006 = 0.086\)\)

    三位居民戴口罩相互独立，求至少有一个人戴口罩的概率

    解： 使用对立事件 \(\(P(\text{至少一人戴口罩}) = 1 - P(\text{三人都不戴口罩})\)\) \(\(= 1 - (1-0.086)^3 = 1 - 0.914^3 ≈ 0.235\)\)

    二、大题部分¶

    题目1：大数定律应用¶

    题目： 随机01字符串，长度1,000,000，估计至少有502,000个1的概率

    解题思路： - 使用中心极限定理 - 设每位为1的概率为0.5 - 期望值：\(\mu = 1000000 \times 0.5 = 500000\) - 方差：\(\sigma^2 = 1000000 \times 0.5 \times 0.5 = 250000\)

    题目2：随机变量的均值与方差¶

    给定条件： \(W = \frac{X_1 + X_2 + X_3 + X_4}{4}\)，其中 \(E(X_i) = 5\)，\(\text{Var}(X_i) = 20\)

    求W的均值和方差

    解： \(\(E(W) = \frac{E(X_1) + E(X_2) + E(X_3) + E(X_4)}{4} = \frac{5 + 5 + 5 + 5}{4} = 5\)\)

    \[\text{Var}(W) = \frac{\text{Var}(X_1) + \text{Var}(X_2) + \text{Var}(X_3) + \text{Var}(X_4)}{16} = \frac{20 + 20 + 20 + 20}{16} = 5\]

    用两种方法计算 \(P[W \geq 20]\)

    方法1： 标准化后使用正态分布 方法2： 切比雪夫不等式

    题目3：联合分布分析¶

    频数分布表：

    Y\X 0 1 2 行总和
    0 160 80 120 360
    1 240 40 160 440
    列总和 400 120 280 800

    概率分布表：

    Y\X 0 1 2 行总和
    0 0.20 0.10 0.15 0.45
    1 0.30 0.05 0.20 0.55
    列总和 0.50 0.15 0.35 1.00

    X的边际分布： - \(P(X=0) = 0.50\) - \(P(X=1) = 0.15\) - \(P(X=2) = 0.35\)

    Y的边际分布： - \(P(Y=0) = 0.45\) - \(P(Y=1) = 0.55\)

    独立性检验：

    检查 \(P(X=i, Y=j) = P(X=i) \times P(Y=j)\) 是否成立：

    • \(P(X=0, Y=0) = 0.20\) vs \(P(X=0) \times P(Y=0) = 0.50 \times 0.45 = 0.225\) ❌

    结论： X和Y 不独立

    计算协方差 Cov(X,Y)：

    \[E[X] = 0 \times 0.5 + 1 \times 0.15 + 2 \times 0.35 = 0.85$$ $$E[Y] = 0 \times 0.45 + 1 \times 0.55 = 0.55\]
    \[E[XY] = \sum_{i,j} ij \cdot P(X=i, Y=j) = 0.45\]
    \[\text{Cov}(X,Y) = E[XY] - E[X]E[Y] = 0.45 - 0.85 \times 0.55 = -0.0175\]

    计算相关系数：

    \[\text{Var}(X) = E[X^2] - (E[X])^2 = 1.55 - 0.85^2 = 0.8275$$ $$\text{Var}(Y) = E[Y^2] - (E[Y])^2 = 0.55 - 0.55^2 = 0.2475\]
    \[\rho_{X,Y} = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)} \sqrt{\text{Var}(Y)}} = \frac{-0.0175}{\sqrt{0.8275 \times 0.2475}} \approx -0.039\]
    题目4：分段概率密度函数¶

    题目： 分段概率密度函数

    \[f(x) = \begin{cases} k(1-x) & , 0 < x < 1 \\ \frac{1}{2}x & , 1 \leq x \leq 2 \\ 0 & , \text{其他} \end{cases}\]

    求参数 k：

    利用概率密度函数的性质：\(\int_{-\infty}^{\infty} f(x)dx = 1\)

    \[\int_0^1 k(1-x)dx + \int_1^2 \frac{1}{2}x dx = 1\]
    \[k\left[x - \frac{x^2}{2}\right]_0^1 + \frac{1}{2}\left[\frac{x^2}{2}\right]_1^2 = 1\]
    \[k\left(1 - \frac{1}{2}\right) + \frac{1}{2}\left(\frac{4}{2} - \frac{1}{2}\right) = 1\]
    \[\frac{k}{2} + \frac{3}{4} = 1\]
    \[k = \frac{1}{2}\]

    求分布函数 F(x)：

    \[F(x) = \begin{cases} 0 & , x \leq 0 \\ \int_0^x \frac{1}{2}(1-t)dt = \frac{1}{4}(2x - x^2) & , 0 < x < 1 \\ \frac{1}{4} + \int_1^x \frac{1}{2}t dt = \frac{1}{4} + \frac{1}{4}(x^2 - 1) = \frac{x^2}{4} & , 1 \leq x \leq 2 \\ 1 & , x > 2 \end{cases}\]

    求 P[X ≥ 1.5]：

    \[P[X \geq 1.5] = 1 - F(1.5) = 1 - \frac{(1.5)^2}{4} = 1 - \frac{2.25}{4} = \frac{7}{16}\]
    题目5：二元连续型随机变量¶

    题目：

    \[f(X,Y) = \begin{cases} \frac{5}{4}x & , y^2 < x < 1 \\ 0 & , \text{其他} \end{cases}\]

    定义域分析：

    积分区域

    • \(x\) 的范围：\((0, 1)\)
    • 对于给定的 \(x\)，\(y\) 的范围：\((-\sqrt{x}, \sqrt{x})\)
    • 约束条件：\(y^2 < x\) 且 \(0 < x < 1\)

    求 P[Y ≤ X]：

    积分区域：\(y \leq x\) 且 \(y^2 < x < 1\)

    对于每个 \(x \in (0,1)\)，\(y \in (-\sqrt{x}, \min(x, \sqrt{x}))\)

    由于 \(x \in (0,1)\) 时，\(\sqrt{x} > x\)，所以 \(y \in (-\sqrt{x}, x)\)

    \[P[Y \leq X] = \int_0^1 \int_{-\sqrt{x}}^x \frac{5}{4}x \, dy \, dx\]
    \[= \int_0^1 \frac{5}{4}x(x + \sqrt{x}) \, dx\]
    \[= \frac{5}{4} \int_0^1 (x^2 + x^{3/2}) \, dx\]
    \[= \frac{5}{4} \left[\frac{x^3}{3} + \frac{2x^{5/2}}{5}\right]_0^1\]
    \[= \frac{5}{4} \left(\frac{1}{3} + \frac{2}{5}\right) = \frac{5}{4} \times \frac{11}{15} = \frac{11}{12}\]

    求边缘概率密度 f_Y(y)：

    对于 \(|y| < 1\)，\(x\) 的范围是 \((y^2, 1)\)：

    \[f_Y(y) = \int_{y^2}^1 \frac{5}{4}x \, dx = \frac{5}{4} \cdot \frac{1}{2}(1 - y^4) = \frac{5}{8}(1 - y^4)\]
    \[f_Y(y) = \begin{cases} \frac{5}{8}(1 - y^4) & , -1 < y < 1 \\ 0 & , \text{其他} \end{cases}\]

    联合概率密度函数：

    题目已经给出了联合概率密度函数 \(f(X,Y)\)，这就是答案：

    \[f(X,Y) = \begin{cases} \frac{5}{4}x & , y^2 < x < 1 \\ 0 & , \text{其他} \end{cases}\]
